# CDFlow

轻量级冷却管理器美化插件，专注简洁与实用。

基于魔兽世界 12.0 内置的**冷却管理器（Cooldown Manager）**系统，CDFlow 可自定义其图标样式、布局与技能激活高亮，让技能冷却展示更符合你的习惯。

---

## 样式定制

- **图标样式**：尺寸、裁剪、边框粗细
- **布局**：居中排列 或 游戏默认对齐
- **每行数量**：自定义每行显示的图标数量
- **间距**：行列间距可调
- **堆叠文字**：层数/充能数字的字体、字号、描边、位置
- **行尺寸覆盖**：为不同行单独设置宽高

## 安装方法

1. 下载并解压到魔兽世界插件目录：
   ```
   World of Warcraft\_retail_\Interface\AddOns
   ```

---

## 使用方法

### 打开设置

- 使用命令：`/cdf`

---

## 依赖

- **魔兽世界**：12.0 及以上版本（Interface 120000+）
- 内置库：LibStub、AceGUI-3.0、LibCustomGlow-1.0、CallbackHandler-1.0
