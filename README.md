# CDFlow


轻量级冷却管理器美化插件，专注简洁与实用。  
基于魔兽世界 12.0 内置的 **Cooldown Manager** 系统，支持图标样式、布局、高亮特效等深度自定义。

---

## 功能概览

- 图标样式：尺寸、裁剪、边框
- 布局控制：居中/默认、间距、每行数量、行尺寸覆盖、追踪状态栏增长方向
- 堆叠数字样式：字号、描边、位置偏移
- 高亮特效：技能激活高亮 + Buff 增益高亮（可独立配置）

## 安装

1. 解压到：
   ```
   World of Warcraft\_retail_\Interface\AddOns\CDFlow
   ```
2. 进入游戏后使用 `/cdf` 打开设置。

## 依赖

- 魔兽世界：12.0+（Interface 120000+）
- 内置库：LibStub、AceGUI-3.0、LibCustomGlow-1.0、CallbackHandler-1.0
